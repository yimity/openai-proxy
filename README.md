## Install wrangler
```dotnetcli
npm install -g wrangler
```

## Install

```sh
npm install  && npm run build
```

## Usage

```sh
npm run deploy
```

## Add custom domain
Add `Route` in `wrangler.toml`

```toml
routes = [
	{ pattern = "Your Custom Domain", custom_domain = true },
]
```
